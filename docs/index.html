What is ffmcp?

ffmcp is a command-line tool inspired by ffmpeg that provides a unified interface for accessing AI services. It allows you to interact with multiple AI providers (OpenAI, Anthropic, etc.) from a single, simple command-line interface.

Just like ffmpeg revolutionized multimedia processing, ffmcp brings the same simplicity and power to AI interactions - making it easy to generate text, analyze images, transcribe audio, and much more directly from your terminal.

Download

Install via pip

pip install ffmcp

Requires Python 3.8 or higher

Install from source

git clone https://github.com/brandonhenry/ffmcp.git
cd ffmcp
pip install -e .

Quick start

# Configure API key
ffmcp config -p openai -k YOUR_KEY

# Generate text
ffmcp generate "Hello, world!"

Quick Examples

Text Generation

ffmcp generate "Write a haiku about coding"

Image Analysis

ffmcp openai vision "What's in this image?" photo.jpg

Audio Transcription

ffmcp openai transcribe audio.mp3

Brains & Memory (Zep)

ffmcp brain create mybrain
ffmcp brain memory add --role user --role-type user --content "Who was Octavia Butler?"
ffmcp brain memory get
